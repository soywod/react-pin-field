# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.0-beta.x | :white_check_mark: |
| 1.1.x   | :white_check_mark: |
| < 1.1.0 | :x:                |

## Reporting a Vulnerability

Open a [new issue](https://github.com/soywod/react-pin-field/issues) and prefix the title with `[security]`.
